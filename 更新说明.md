## 🎯 功能说明

### 累计总打卡次数
- **计算逻辑**：所有记录的总数
- **更新时机**：每次打卡后立即更新
- **数据来源**：localStorage中的所有记录
- **动画效果**：数字放大缩小的脉冲动画

### 连续打卡天数
- **计算逻辑**：
  1. 从今天开始往前检查
  2. 如果这一天有打卡记录，连续天数+1
  3. 如果这一天没有记录（且不是今天），中断计数
  4. 最多回溯365天

- **示例**：
  - 今天有打卡：1天
  - 昨天也有打卡：2天
  - 前天也有打卡：3天
  - 大前天没打卡：中断，只算3天

- **特殊情况**：
  - 今天还没打卡，也会从昨天开始算连续天数
  - 跨月、跨年都能正确计算

### 数据持久化
- 所有数据保存在 localStorage
- 每次打卡自动保存
- 关闭应用后数据不会丢失
- 支持历史记录查询

## 📊 数据统计

### 当前显示的数据
1. **今日目标**：2000ml（固定值）
2. **今日打卡**：今天的打卡次数
3. **还需饮用**：2000ml - 今日已喝量
4. **累计总打卡**：所有历史打卡总数（新增）
5. **连续打卡**：连续打卡的天数（新增）

### 数据计算公式
```javascript
今日打卡次数 = 今天的记录数量
今日已喝量 = 今天所有记录的amount总和
还需饮用 = max(2000 - 今日已喝量, 0)
累计总打卡 = 所有历史记录的数量
连续打卡天数 = 从今天往前推，有连续记录的天数
```
