# 在Mac上快速构建

## 🚀 超快速构建（3步）

```bash
# 1. 进入项目目录
cd RUthirsty-cordova

# 2. 运行构建脚本
./build-mac.sh

# 3. 选择选项2（构建DMG）
# 等待构建完成...
```

## 📦 构建结果

```
dist-mac/RUthirsty-1.0.0.dmg
```

双击DMG文件，拖拽到Applications即可安装！

## 🎯 快速测试

如果只想快速看看效果，不生成安装包：

```bash
./build-mac.sh
# 选择选项1（快速测试）
```

应用会直接启动，看到毛玻璃风格的打卡界面。

## 📖 详细说明

查看完整文档：[Mac应用构建指南.md](./Mac应用构建指南.md)

## ⚠️ 首次运行可能提示安全警告

解决方法：
```bash
# 在终端运行这个命令
xattr -cr /Applications/RUthirsty.app
```

或者：
1. 右键点击应用
2. 选择"打开"
3. 点击"打开"确认

## 💡 提示

- 构建时间：首次约5-10分钟（需下载Electron）
- 文件大小：约100-150MB（包含完整浏览器引擎）
- 系统要求：macOS 10.13或更高
